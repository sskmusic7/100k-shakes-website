The website uses gradient placeholders for images. To add real images:

1. Place images in the `images/` directory
2. Recommended sizes:
   - Hero images: 1920x1080px minimum
   - Product images: 800x800px
   - Story images: 1200x800px
   - Social feed: 400x400px (square)

3. Replace gradient backgrounds in HTML with:
   ```html
   <div class="shake-image" style="background-image: url('images/hero-image.jpg');"></div>
   ```
