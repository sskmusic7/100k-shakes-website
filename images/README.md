# Images Directory

This directory is for storing website images.

## Recommended Image Sizes

- **Hero images**: 1920x1080px minimum
- **Product images**: 800x800px (square)
- **Story images**: 1200x800px
- **Social feed images**: 400x400px (square)
- **Category icons**: 200x200px

## Image Placeholders

The website currently uses CSS gradients as placeholders. To replace them:

1. Add your images to this directory
2. Update the HTML files to reference the image paths
3. Replace gradient backgrounds with `background-image: url('images/your-image.jpg')`

## Image Optimization

- Use WebP format for better performance
- Compress images without losing quality
- Include @2x versions for retina displays
- Add descriptive alt-text for accessibility
